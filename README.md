# POOSD-Cartridge
Group 14's project for Processes and Object Oriented Software Development at the University of Central Florida.

## What is it?
Cartridge is a game library tracker inspired by Goodreads, built for gamers who want to organize and keep track of their gaming experiences. Users can log games they've played, build custom lists, and write reviews and ratings for their collection.

## Features
- User registration and login with email verification
- Create, organize, and manage custom game lists (e.g. "Playing", "Want to Play", "Completed")
- Add games to your library with details such as hours played, rating, review, and platform
- Game data is automatically pulled and cached from the RAWG API
- Each game entry is private to the user

## Project Specifications
- Web app built with React, backend API built with Node.js and Express
- Mobile app built with React Native, targeting iOS and Android
- Data is stored in a MongoDB database
- The frontend communicates with the backend through a REST API
- Hosted on DigitalOcean
